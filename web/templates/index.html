Live GPS Tracker

The GPS tracker creates a local web server to demonstrate location data capture.

⚠️ Educational Use Only: This feature requires the target to click a link and grant location permission. Do not use without consent.

How to Use

1. Start the server from the terminal:

python3 GhostTrackerPro.py --gps

2. Share the generated link with the target (local network or via Ngrok).

3. When they grant location permission, coordinates appear in your terminal and are saved to logs.
